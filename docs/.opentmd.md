# OpenTMD-Cli 项目指令

## 技术栈

- **语言**: Go 1.22+
- **CLI 框架**: [Cobra](https://github.com/spf13/cobra)
- **TUI 框架**: [Bubble Tea](https://github.com/charmbracelet/bubbletea) + [Lip Gloss](https://github.com/charmbracelet/lipgloss)
- **配置格式**: TOML (via [go-toml/v2](https://github.com/pelletier/go-toml))
- **语义分析**: [go-tree-sitter](https://github.com/smacker/go-tree-sitter) (Go/Rust/Python/JS/TS/Java/C/C++/C#/PHP)
- **构建**: `CGO_ENABLED=0 go build -trimpath -ldflags "-s -w"`

## 目录结构

```
cmd/opentmd/       — CLI 入口（Cobra 命令树）
internal/          — 内部包（不对外导出）
  agent/           — Agent 编排器（多步循环、prompt 构建）
  config/          — 配置管理（TOML 读写、Provider 预设）
  daemon/          — HTTP Daemon 服务（SSE 聊天、LSP/MCP 状态）
  deepseek/        — OpenAI 兼容 HTTP 客户端
  diagnostics/     — LSP 实时诊断
  filehistory/     — 文件修改历史（/undo）
  fileio/          — 文件读写
  glob/            — 文件模式匹配
  graph/           — Go 代码图谱索引
  grep/            — 正则搜索
  hook/            — Hooks 系统（PreToolUse/PostToolUse/UserPromptSubmit）
  llm/             — LLM 类型定义与 Provider 接口
  lsp/             — LSP 客户端（gopls/rust-analyzer/tsserver/pylsp）
  mcp/             — MCP 客户端（stdio/SSE）
  memory/          — 持久记忆系统（memory.md）
  pathutil/        — 路径解析工具
  permission/      — 权限审批（AutoApprove/RequireApproval）
  pricing/         — Token 费用估算
  project/         — 项目指令加载、技术栈检测
  provider/        — Provider 工厂
  semantic/        — 多语言代码符号分析（tree-sitter queries）
  session/         — 会话持久化（JSON 文件）
  setup/           — 首次启动向导
  shell/           — Shell 命令执行
  skill/           — Skills 系统（SKILL.md 模板）
  tool/            — 工具注册与执行（read/write/edit/grep/glob/bash 等）
  trust/           — 工作目录信任管理
  tui/             — Bubble Tea TUI（聊天界面、斜杠命令、模态框）
  web/             — 网络工具（web_search / web_fetch）
extensions/        — IDE 扩展
docker/            — Docker 部署
scripts/           — 构建、安装、部署脚本
docs/              — 文档
```

## 构建与测试

```bash
go build -o opentmd ./cmd/opentmd/             # 构建
go build ./...                                  # 检查编译
go test ./...                                   # 全部测试
go test ./internal/memory/... -v                # 单包测试
```

## 代码规范

- 包名小写、无下划线
- 错误处理优先（不忽略 err）
- 工具 Handler 签名: `func(ctx context.Context, args json.RawMessage) (string, error)`
- 使用 `json:"snake_case"` 标签
- 新增功能需同步更新 docs/
